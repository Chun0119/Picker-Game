Unity Version 2019.4.30

To run the game in Unity Editor:
1. Go to Assets/Scenes/Main
2. Play

The game should be run in landscape ratio.

To change any configuration of the picker game:
1. Go to Assets/Scripts/ScriptableObjects/PickerConfig
2. Modify the numbers you want
*The maximum number of options is limited to 9 due to the UI constraint.
